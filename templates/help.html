WELCOME!

This is where you get help on how to use this application!

This application helps you to:

  • Search and find any books and their information
  • Generate a random book with custom book type
  • Add new books with their full information to the database
  • Update existing books
  • Delete existing books


1. Search and find books

This feature helps you to find any book in this library if it was added to the database. (see #3 on how to add new book). to search for a book, go to this page. you can search a book by its name, author or book type. when you enter your search text on the input field and click the search icon button or enter, your search results will be displayed there. To see book information, click on any book that is shown there. click again to hide the information.

2. Generate random book

This is where you can generate new random book from the database with custom book type. To generate new random book, go to this and choose book type. then click on the 'generate random book' button.

3. Add new book

If you want to add new book to this application, go to this and fill all needed information responsibly then click on 'submit' button. after you clicked the submit button, the information you just entered will be accessible for any user.

4. Update existing books

If you think the information of a book is outdated and needs to be updated, you can do this here. you can edit some part of the information or the whole part. to update a book go to this page and search for the book you want to update. then select the book from search results. fill all needed information and click submit

4. Delete a book

If you think a book is no more useful in the database or if it doesnt exist anymore in the library, you can remove it from the database (PLEASE BE CAREFUL WHEN DELETING BOOKS! THIS ACTION CAN NOT BE UNDO!)
